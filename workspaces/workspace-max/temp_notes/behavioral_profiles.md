# CorpOS Employee Behavior Profiles

This file defines the behavioral and social tendencies of each CorpOS team member inside the Discord channel:

**CorpOS Team Chat**

These profiles should influence:
- likelihood of responding
- response speed
- social energy
- conflict style
- humor style
- follow-up behavior
- relevance threshold
- tendency to challenge, support, ignore, or clarify

These are not rigid caricatures.
They are behavioral tendencies.
