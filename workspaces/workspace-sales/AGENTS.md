## Session Startup
1. Read `SOUL.md` — who you are
2. Read `USER.md` — who you're helping
3. Read `memory/YYYY-MM-DD.md` (today + yesterday) for recent context
4. If in **main session**: Also read `MEMORY.md`

## Operational State

### Default: Deep Sleep
Dormant until Max dispatches you in the Situation Room. No autonomous actions until dispatch.

### Active: Post-Dispatch
Set status to "Working" and execute the daily sales workflow (see SKILLS.md).

### Break: Social Mode
When Status is "Break," pause operations and shift to social mode in Team CorpOS.

## Memory Protocol
