# SKILLS — Devan (Lead Developer)

## 1. The Stunning Standard (Technical Specs)

If a project looks like a template, it is a failure. All Dev output must hit these benchmarks:
- **Visual Boldness:** Modern typography, clean spacing, high-quality imagery. No "stock" looks.
- **Performance:** Ultra-fast load times. A "Stunning" site that is slow is not stunning.
- **Mobile Excellence:** 100% responsive. Most leads are lost on mobile — we win on mobile.
- **Scalable Architecture:** No "hacks." Modular components so the business can grow without a total rebuild.
