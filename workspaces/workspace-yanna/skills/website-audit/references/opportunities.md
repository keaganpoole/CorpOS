# Opportunity Tags

Use these tags when identifying issues. Be specific in the Discovery description — the tag is a label, the description is the evidence.

## Design Issues

| Tag | When to use |
|-----|------------|
| `Outdated Design` | Site looks 5+ years old. Dated colors, fonts, layout conventions |
| `Poor Visual Hierarchy` | Can't tell what's important. No clear focal points |
| `Low Quality Images` | Blurry, pixelated, stretched, or stock photos that don't match the business |
| `Inconsistent Branding` | Mixed fonts, clashing colors, no logo or different logos on different pages |

## Mobile Issues

| Tag | When to use |
|-----|------------|
| `Not Mobile Friendly` | Layout breaks on mobile. Text overlaps, elements cut off |
| `Small Tap Targets` | Buttons/links too small to tap accurately on phone |
| `Horizontal Scroll` | Page wider than viewport, requires horizontal scrolling |

## Functionality Issues

| Tag | When to use |
|-----|------------|
| `Poor Functionality` | Broken links, forms that don't submit, features that don't work |
| `Slow Loading` | Page takes 5+ seconds to load, unoptimized images |
| `Broken Links` | 404 errors, links to dead pages |
| `No SSL` | HTTP only, browser shows "Not Secure" warning |

## Content Issues

| Tag | When to use |
|-----|------------|
| `Limited Features` | Missing expected features for the industry (no menu for restaurant, no services page for contractor) |
| `Missing Contact Info` | No phone, email, or address visible anywhere |
| `Thin Content` | One page, barely any text, placeholder content |
| `No About Page` | No information about the business, owners, or history |

## Conversion Issues

| Tag | When to use |
|-----|------------|
| `No Clear CTA` | No call-to-action. Visitor doesn't know what to do next |
| `No Contact Form` | No way to reach them except clicking a mailto: link |
| `No Online Booking` | Industry-standard booking/ordering not available |

## Existence Issues

| Tag | When to use |
|-----|------------|
| `No Website` | Business has no website at all |
| `Expired Domain` | Domain registration expired, shows parking page |
| `Under Construction` | Placeholder page, "coming soon" |

## Using Tags in Discovery

Always pair tags with specific evidence:

**Good:**
```
- Outdated Design: Site uses a dark purple gradient background with serif fonts and a Flash-based header animation
- Not Mobile Friendly: Navigation menu overlaps body text at 375px width, images overflow viewport
- No Clear CTA: Phone number only visible in footer, no contact form on any page
```

**Bad:**
```
- Outdated Design: looks old
- Not Mobile Friendly: bad on phone
- No Clear CTA: needs better CTA
```
